---
description: Create screen design components
---

1. Identify the section and spec.
2. Create `src/sections/[section-slug]/components/` directory if needed.
3. Create React components in that directory using Tailwind v4.
4. Ensure components export props interface.
5. Create `src/sections/[section-slug]/[ViewName].tsx` to preview the component with sample data.
6. Notify user to check the preview in the app.
