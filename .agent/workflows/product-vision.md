---
description: Define the product vision and core requirements
---

1. Check if `product/product-overview.md` exists.
2. If it exists, ask the user if they want to update it.
3. If not, ask the user for:
   - Product Name
   - One-sentence description
   - 3 Key problems it solves
   - 3 Key features
4. Create `product/product-overview.md` with the following format:

   ```markdown
   # [Product Name]

   ## Description

   [Description]

   ## Problems & Solutions

   ### Problem 1: [Problem Title]

   [Solution]

   ### Problem 2: [Problem Title]

   [Solution]

   ### Problem 3: [Problem Title]

   [Solution]

   ## Key Features

   - [Feature 1]
   - [Feature 2]
   - [Feature 3]
   ```

5. Notify the user that the vision is set.
